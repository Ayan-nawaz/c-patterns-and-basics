# C Patterns and Basics

A collection of beginner-friendly C programs for practicing loops, pattern printing, and basic mathematical operations.

## 📂 Programs Included

### 1. power.c
- Calculates the power of a number using C.

### 2. patternprinting.c
- Prints basic patterns using nested loops.

### 3. notriangle.c
- Prints a number triangle pattern.

### 4. oddnotriangle.c
- Prints an odd number triangle pattern.

### 5. startriangle.c
- Prints a star (`*`) triangle pattern.

## 📚 Concepts Covered

- Variables and Data Types
- Input and Output
- For Loops
- Nested Loops
- Pattern Printing
- Basic Mathematics

## 💻 Language

- C Programming

## 🛠️ Tools Used

- Visual Studio Code
- GCC Compiler

## ▶️ How to Run

Compile the program:

```bash
gcc filename.c -o output
```

Run the program:

```bash
./output
```

## 🎯 Purpose

This repository is created to improve C programming fundamentals by practicing pattern printing and basic programming problems.
